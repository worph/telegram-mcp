Telegram MCP

What is this?

Telegram MCP is a bridge between Telegram and any MCP-compatible server. Messages you send to the bot get forwarded as tool calls, and the response comes back as a reply.

Example uses
  • Echo test — Use the built-in echo tool to verify everything works. Messages you send come right back.
  • Claude Code — Point it at a claude-code-container to run Claude from Telegram, with permission prompts for file writes and shell commands.
  • Any MCP server — Connect to any tool on any MCP endpoint. Map Telegram message fields to tool parameters with {{text}}, {{chatId}}, etc.
Quick start
  1. Create a bot via @BotFather and paste the token below
  2. Pick a preset (Echo to test, or Claude Code Container) or enter a custom MCP endpoint
  3. Send a message to your bot on Telegram
